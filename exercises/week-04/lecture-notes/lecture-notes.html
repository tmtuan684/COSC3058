Position, Box Model, Layout

Position

CSS property position controls the position of HTML element in an HTML pages.
    Position has 5 possible values:
  • static: default value. With this value, an HTML elements appears in the natural flow orders of HTML elements - which element declared first, appears first.
  • relative: With this value, an HTML elements appears relative to its "static" position by distance indicated by property top, left, right, bottom
  • absolute: an element is freed from natural flow of a page, is positioned relatived to its nearest ancestor which has position style (with values other than default/static) or relative to its contained block (if it has not positioned ancestors)
  • fixed: an element is freed from natural flow of a page, is positioned relatived to its containing element
  • sticky: an HTML elements appears in the natural flow orders of HTML elements, offset relative to the nearest scrolling ancestor.

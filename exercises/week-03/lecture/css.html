Basic CSS Selectors

Element, Class, Id

Element is of HTML element, e.g., h1, h2, p, div, span, body

Class gathers a collection of elements which have things in common

Id is a unique identifier for an element, distinguishing an element from others semantically


/* Element Selector */
element {
    attribute_1: value;
    attribute_2: value;
    ...
    attribute_n: value;
}

/* Class Selector */
.class_name {
    attribute_1: value;
    attribute_2: value;
    ...
    attribute_n: value
}

/* Id Selector */
#id_name {
    attribute_1: value;
    attribute_2: value;
    ...
    attribute_n: value
}
Example: Element, Class, Id CSS selector

Hierarchy

Parent


/* Descendant Selector: element that is contained in another element regardless how deep in level it is   */
parent_element child_element {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* Child Selector: element that is a direct child of another element */
parent_element > child_element {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}
Example: Hierarchy CSS selector

Sibling (Brothers and Sisters)


/* Sibling Selector: element at the same level of a specific element */
A ~ B {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* Adjacent Sibling Selector: element right next to a specific element of the same level */
A + B {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}
Example: Sibling CSS selector

Pseudo

Pseudo‐class


/* Pseudo-class selector: element at a specific state */
element:[pseudo-class] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}

Pseudo‐element


/* Pseudo-element selector: element in a specific relationship to other element of same type */
A::[pseudo-element] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}
Example: Pseudo-class and Pseudo-element CSS selector

Attribute selectors


/* select element having [attribute] */
element[attribute] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* select elements whose attribute has exact value */
element[attribute="value"] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* select elements whose attribute contains the "word" in separation from other words  */
element[attribute~="word"] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* select elements whose attribute contains the "value" within a string  */
element[attribute*="value"] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* select elements whose attribute begins with the value  */
element[attribute^="value"] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}


/* select elements whose attribute ends with the value  */
element[attribute$="value"] {
   attribute_1: value;
   attribute_2: value;
   ...
   attribute_n: value;
}

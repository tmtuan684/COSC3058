Browser's Viewport

A browser's viewport is an browser windows being currently viewed. More specifically, the viewport is the part of the document, or html page, that is currently visible. Viewport is determined relatively to its container.

<meta name="viewport" content="width=device-width" /> means that

the size of the viewport is preferably set to the device-width; CSS rules refers to width of the device on which webpage is currently displayed to present content.

Media Query

Media Query is CSS technique of applying different CSS rules in accordance to screen size, orientation, or device type.
